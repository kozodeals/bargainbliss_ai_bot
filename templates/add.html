Common Message Key Suggestions
General Messages:
  • welcome_message - Bot welcome message
  • help_text - Help information
  • error_general - General error message
  • success_message - Success confirmation
Command Responses:
  • start_command - /start command response
  • help_command - /help command response
  • tips_command - /tips command response
Error Messages:
  • error_invalid_input - Invalid input error
  • error_not_found - Resource not found
  • error_permission - Permission denied
User Guidance:
  • instruction_step1 - First step instruction
  • instruction_step2 - Second step instruction
  • confirmation_question - Ask for confirmation
